# Focus Productivity

A standalone productivity web app with:

- Synced categories across tasks, habits, and planner notes
- Daily, weekly, and monthly planning spaces
- Weekly tasks can be assigned to specific weekdays
- To-do filters by period or category
- Habit tracking with daily check-ins
- Theme customization with pink, blue, black, green, and sunset palettes
- Local browser storage, so your workspace stays saved on this device

Open `index.html` in a browser to use the app. No build step or install is required.
